# Install G-Drive CLI

**To install G-Drive CLI:**

**1.** Login as root<br>
**2.** Copy these Lines and Paste into Terminal
```bash
wget https://github.com/cachecleanerjeet/install-gdrive-cli/raw/master/install.sh
chmod +x install.sh
bash install.sh
```
